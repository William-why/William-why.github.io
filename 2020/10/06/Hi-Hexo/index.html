Hexo + GitHub Pages 搭建个人博客：参考来源

问题总结：

  1. 无法访问 github.io

    • https://tool.chinaz.com/dns/ 对 yourUsername.github.io 进行 DNS 查询，选择延时相对较小的且稳定的 ip

    • 在C:\Windows\System32\drivers\etc中找到 hosts 文件

    • 以管理员身份打开 PowerShell ，使用notepad hosts命令记事本修改 hosts 文件

  • 在 hosts 文件末尾新添：ip yourUsername.github.io

  • ipconfig /flushdns命令刷新DNS缓存
